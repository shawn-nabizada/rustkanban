RustKanban

A fast, keyboard-driven kanban board for your terminal

Features

Vim-style Navigation

Navigate columns with H/L, tasks with arrow keys. Everything is keyboard-first.

Cross-machine Sync

Log in with GitHub and sync your board across all your devices.

Tags & Search

Organize with tags, filter by tag, search across all tasks.

Install

curl -fsSL https://rustkanban.com/install.sh | sh

Or with cargo:

cargo install --path crates/rk-client

Sync Setup

rk login

Opens your browser for GitHub authentication. Your board syncs automatically on open and close.
